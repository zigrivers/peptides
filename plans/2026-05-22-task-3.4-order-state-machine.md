## State Machine
```
DRAFT → SENT → CONFIRMED → PAYMENT_SENT → RECEIVED (terminal)
Any non-terminal (DRAFT | SENT | CONFIRMED | PAYMENT_SENT | STALE) → CANCELLED (terminal)
SENT (age > 14 days) → STALE (via cron)
```
Forward-only invariant: non-cancel transitions cannot move backward.

### Step 2: Implement `cancelOrder` in OrderService

```typescript
export async function cancelOrder(userId: string, orderId: string): Promise<void>
```

- Fetch order: `findFirst({ where: { id: orderId, userId } })` — throw `order_not_found` if null
- Terminal check: if status is `RECEIVED` or `CANCELLED` throw `invalid_order_transition`
- `withAudit` wrapping `updateMany({ where: { id: orderId, userId, status: { in: NON_TERMINAL_STATUSES } }, data: { status: 'CANCELLED', cancelledAt: now, cancelledByUserId: userId } })`
- Check `count === 0` → throw `invalid_order_transition`
- Audit: `ORDER_CANCELLED`, `oldValues: { status: order.status }`, `newValues: { status: 'CANCELLED' }`

Where `NON_TERMINAL_STATUSES = ['DRAFT', 'SENT', 'CONFIRMED', 'PAYMENT_SENT', 'STALE']`

### Step 3: Implement `markOrdersStale` in OrderService

```typescript
export async function markOrdersStale(now: Date): Promise<number>
```

- Compute `cutoff = new Date(now.getTime() - 14 * 24 * 60 * 60 * 1000)`
- Find all SENT orders with `sentAt < cutoff` and `staleFlaggedAt` IS NULL: `findMany`
- For each order: `withAudit` wrapping `updateMany({ where: { id, status: 'SENT' }, data: { status: 'STALE', staleFlaggedAt: now } })`
- Audit: `ORDER_MARKED_STALE`, `oldValues: { status: 'SENT' }`, `newValues: { status: 'STALE' }`, `actorUserId: order.userId`
- Return total count staled

### Step 6: Cron endpoint `POST /api/cron/stale-orders`

```typescript
// app/api/cron/stale-orders/route.ts
import { NextResponse } from 'next/server';
import { markOrdersStale } from '@/lib/ordering/application/OrderService';

export async function POST(req: Request) {
  const secret = req.headers.get('authorization')?.replace('Bearer ', '');
  if (secret !== process.env.CRON_SECRET) {
    return NextResponse.json({ error: 'Unauthorized' }, { status: 401 });
  }
  const count = await markOrdersStale(new Date());
  return NextResponse.json({ staled: count });
}
```
